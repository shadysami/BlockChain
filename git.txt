- git crash course 


= git clone <repository-url> 
      - to clone a repo from a remote source on github on your local machine 


= git add <file-name> // stage certain file/s 
= git add . // to stage all changes in the current directory 
   - to add file to the stage before being commited 

- after adding changes to the stage , you will need to commit every change with a declaring message for easily    identify what did you change .

        = git commit -m "Your commit message"

    
>> After committing changes locally, you can upload them to a remote repository (e.g., GitHub).
      = git push origin <branch-name>


>> To update your local repository with changes from the remote repository, use git pull.
      = git pull origin <branch-name>
